À RETRAVAILLER

Commentaire

L'imparfait se forme avec le radical de « nous » au présent + les terminaisons -ais, -ais, -ait, -ions, -iez, -aient.

Exemple : nous jouons → je jouais. Vous avez confondu les terminaisons de l'imparfait avec celles du présent.

Corrections détaillées

1. j'étai → étais

1re personne → terminaison -ais

2. il pleuvais → pleuvait

3e personne → terminaison -ait

3. vous prennez → preniez

radical de « nous prenons » → pren- + -iez

4. parents lisait → lisaient

pluriel → terminaison -aient
